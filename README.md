# Prerequisite
- PHP 7.4.9
- Composer

# Install
1. Copie the project
   ```
   git clone https://github.com/Oberson-Gael/Gob-Groovy.git
   ```
2. Insttall the composer
   ```
   composer i
   ```
4. Give your API Youtube key in the file `index.php`
5. Run this command for run websocket server
   ```
   php app.php
   ```
6. Run your server php
